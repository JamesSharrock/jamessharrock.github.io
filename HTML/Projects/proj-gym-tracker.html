Gym Tracker - Console

Features:

  • Allows a user to create(save), load and delete their workouts.

  • Calculates which plates are required on a barbell.

  • Estimates user's One Rep Max.



This project was created using C# .NET. The code structure is as follows -


  • Helper Libraries:

    • helper-library.cs: Stores all possible colours and has various methods for printing colour outputs to the console.

  • Classes:

    • Exercise.cs: A singular exercise. Stores Name, Weight, Sets and Reps.

    • Log.cs: A singular log. Stores Name, Date and a list of Exercise objects.
