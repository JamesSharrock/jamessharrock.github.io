MY GCSE Programming Project

For my GCSE Programming Project, I undertook the second task from the OCR exam board. This was described as a 2-player dice game. The purpose of the game was to allow multiple rounds, calculate scores and produce a winner. The entire criteria is listed below:



Design, develop, test and evaluate a program that:

- Allows each player to roll two 6-sided dice.
- Calculates and outputs the points for each round and each player's total score.
- Allows the players to play 5 rounds.
- If both players have the same score after 5 rounds, allows each player to roll 1 die each until someone wins.
- Outputs who has won at the end of the 5 rounds.
- Stores the winner's score, and their name, in an external file.
- Displays the score and player name of the top 5 winning scores from the external file.



I immediately decomposed the criteria into a set of grouped requirements. I started programming these requirements by doing everything sequentially. I created a single round between two players with a scoring system to calculate who would win. Once I was satisfied with this I created the framework for the rest of my functions. The overall code struture was broken down into the following functions:

- Intro():
- Rules():
- Roll():
- Dice():
- Round():
- ScoreHandling():
- Decider():
- Total():
- Stats():
- Game():
